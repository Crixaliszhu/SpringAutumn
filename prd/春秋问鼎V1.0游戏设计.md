《春秋问鼎》V1.0 MVP 游戏设计

    
    一、设计目标

    V1.0 目标不是完整模拟春秋天下，而是在微信小游戏环境中验证核心玩法：

    从边境流民村开始，通过经营、征兵、战争和扩张，最终成长为一方诸侯。

    核心循环：

    人口 → 土地产出 → 税收 → 积累粮钱 → 建设 → 征兵 → 战争 → 占领 → 增加人口 → 更大规模扩张

    2.2 玩家与中立势力

    玩家不属于任何国家。

    玩家初始：

    1座边境流民村

    地图额外存在：

    1座中立村；

    1个中立边境郡（包含3座村庄）。

    全地图总村庄：68座。

    

    
    
    
    三、玩家成长阶段

    
    第一阶段：流民村

    初始状态：

    户数：30户；

    人口：150人；

    土地：1500亩；

    粮食：50000斤；

    铜钱：300；

    士兵：10人；

    民心：90。

    玩家在此阶段不会被任何国家视为威胁。

    规则：

    国家不会攻击玩家；

    玩家可以自由发展；

    可以占领中立村和中立边境郡。

    

    
    
    第二阶段：地方势力

    条件：

    拥有一个完整郡级区域或占领国家城池。

    此时：

    正式进入外交系统；

    国家开始评估玩家威胁；

    可能出现招揽、联盟、战争等事件。

    

    
    
    第三阶段：诸侯争霸

    拥有多个城池后，进入天下争霸阶段。

    目标：

    击败秦、晋、齐、楚；

    控制洛邑；

    问鼎天下。

    

    
    
    
    四、时间与结算系统

    游戏采用实时运行模式，不采用传统回合制。

    世界持续运行。

    所有经济、人口、军事维护等数据：

    每游戏月结算一次。

    主要结算：

    粮食税收；

    铜钱税收；

    士兵军粮消耗；

    士兵军饷支付；

    人口增长；

    建筑完成；

    AI 国家决策。

    

    
    
    五、人口、土地与农业系统

    基础规则：

    1户 = 5人；

    每户拥有50亩土地；

    每亩年产粮100斤。

    因此：

    每户年产粮：

    50 × 100 = 5000斤。

    

    
    
    六、税收系统

    
    粮食税

    税率：

    15%。

    因此：

    每户每年缴纳：

    750斤粮食。

    折合每月：

    约62斤。

    

    
    
    铜钱税

    每人每年缴纳：

    2铜钱。

    因此：

    每户（5人）：

    10铜钱/年。

    折合：

    约0.83铜钱/月。

    

    
    
    
    七、军事系统

    V1.0 不设计复杂兵种。

    全世界只有一种士兵。

    
    士兵属性

    粮食消耗：3斤/天；

    每月粮食消耗：90斤；

    军饷：1铜钱/月；

    战斗力：1。

    

    
    
    
    八、守军与野战军

    
    守军

    所有村庄和城池拥有常驻守军。

    和平时期：

    守军不离开驻地；

    国家不存在野战军；

    地图上没有国家移动军队。

    中立势力和周王室：

    永远不会主动组建野战军。

    

    
    
    战争状态

    国家发动战争时：

    国君从各地抽调守军。

    流程：

    守军 → 抽调 → 组成野战军 → 任命将领 → 出征

    一个国家同时最多存在：

    1～3支野战军。

    战争结束后：

    野战军解散，士兵返回原驻地。

    十一、国家 AI 规则

    AI 每月进行一次战略判断。

    和平时期：

    发展经济；

    建设；

    增加人口；

    保持驻军。

    不会主动维持野战军。

    战争触发条件：

    本国被攻击；

    与敌国关系恶化；

    实力明显超过邻国。

    满足条件后：

    抽调守军；

    组建1～3支野战军；

    发动战争。

    

    
    
    十二、V1.0 游戏节奏目标

    0～30分钟：

    玩家经营流民村，占领附近中立村。

    30～90分钟：

    统一中立边境郡，成为地方势力。

    1～3小时：

    挑战国家边境城市。

    3小时以后：

    进入诸侯争霸阶段。

    

    V1.0 设计原则：

    小规模地图、简单经济、低计算量、高成长反馈。

    优先验证“从一村崛起，最终问鼎天下”的核心体验。
